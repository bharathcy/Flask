Test results for Dry and Formal Runs:

This tool captures the code coverage metric for a function under test in tabular format for the coverage metric present in each VECTORCAST generated test result html files.

  • Collection of the function name(s), file name(s), coverage and PRs information from test case files, test script and result files for creating structural coverage matrix
  • Collection of the PR reference referred in Test Cases and Test Scripts



Procedure:

  • Place the list of test result .html files in one folder along with script bat file” Coverage.bat “along with Coverage.py script file.
  • Run the Coverage.bat file.



Output:

Test result reports the consolidated coverage matrix report is generated in tool Output file data.csv.
